# Screenshot Guide

Recommended screenshots for the GitHub README:

1. `api-docs.png` — FastAPI Swagger UI showing `/health` and `/predict`.
2. `predict-endpoint.png` — `/predict` request/response example.
3. Optional: `training-report.png` — generated training accuracy plot from `reports/training_accuracy.png`.
4. Optional: `tests-passing.png` — terminal output showing `pytest -q` and `ruff check app src tests` passing.

Avoid committing private face images, webcam captures, or copyrighted sample photos. Use neutral/demo images only when you have usage rights.
